# My AI Skills Portfolio

这是一个个人 AI 工作流项目，用来保存我最常用的 4 个 Skill。

这里的 Skill 不是“我发明了一个 AI 模型”，而是把我反复使用的 AI 工作方法写成可复用的 `SKILL.md` 文件。它可以作为：

- 给 ChatGPT / Codex / Copilot 使用的提示词模板
- GitHub 上可展示的 AI 工作流作品
- 个人作品集里证明“我能系统使用 AI 工具”的项目
- 以后继续扩展成个人 AI 助手/工作流系统的基础

## 我现在做的 4 个 Skill

| Skill | 用途 |
|---|---|
| 简历英文翻译 Skill | 英文简历翻译与润色，Times New Roman、英文地道、求职导向、保留量化成果 |
| 每日财经简报 Skill | 按“重要新闻—趋势信号—可写选题—继续追问”整理财经/AI/媒体动态 |
| 微信表情包处理 Skill | 固定记住微信表情包尺寸、透明背景、暗夜模式、压缩要求 |
| 学术论文修改 Skill | 学术但不僵硬、引用不能乱配、逻辑清楚、少 AI 腔 |

## 文件在哪里

GitHub / Copilot 风格的 Skill 文件放在：

```text
.github/skills/
  resume-english-translation/SKILL.md
  daily-finance-briefing/SKILL.md
  wechat-sticker-processing/SKILL.md
  academic-paper-review/SKILL.md
```

每个 `SKILL.md` 里都写了：

- 什么时候使用
- 核心目标
- 固定输出格式
- 工作流程
- 注意事项
- 避免什么错误

## 如何使用

### 最简单方式

打开任意 `SKILL.md`，复制里面的内容，粘贴给 ChatGPT / Codex，然后再说你的具体任务。

### 给 Codex / Copilot 使用

如果你的工具支持读取仓库里的 skill，可以让它使用对应 skill，例如：

```text
请使用 .github/skills/academic-paper-review/SKILL.md 的规则，帮我微改下面这段论文。
```
