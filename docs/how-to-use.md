# 怎么使用这 4 个 Skill

## 你现在要理解的一句话

Skill 不是神奇插件，而是“固定工作方法”。

你把 Skill 放在 GitHub 上，有两个用处：

1. 自己以后可以复制使用，减少反复解释。
2. 作为作品集，展示你会把 AI 工作流标准化。

## 使用方法 1：复制粘贴

打开某个 `SKILL.md`，复制全部内容，粘贴给 ChatGPT，然后说：

```text
请按这个 Skill 的规则处理下面的任务：
...
```

## 使用方法 2：让 Codex 读仓库

如果 Codex 能访问你的仓库，你可以说：

```text
请读取 .github/skills/resume-english-translation/SKILL.md，
以后处理我的英文简历时都按这个规则。
```

## 使用方法 3：作为作品集项目

你可以在个人网站或简历里写：

```text
AI Skills Portfolio
Designed four reusable AI workflow skills for resume translation, financial briefing, WeChat sticker processing, and academic writing review. The project demonstrates prompt engineering, workflow standardization, and practical AI tool use across writing, research, and content production.
```

## 不要误解

这个项目不是：

- 不是你训练了一个 AI 模型
- 不是你发布了官方 ChatGPT Skill
- 不是一个复杂软件系统

它是：

- 你把 AI 使用方法整理成标准流程
- 你用 GitHub 做项目化展示
- 你为未来用 Codex/Copilot 做项目打基础
